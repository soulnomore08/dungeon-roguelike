# Dark Dungeon Roguelike

A dark fantasy side-scrolling roguelike built with Phaser 3 and Vite.

## Current Features
- 10 handcrafted levels
- Level progression map
- Save system
- Multiple enemy types
- Boss fights
- Weapon switching
- Dash mechanics

## Planned Features
- Sound effects
- Background music
- Additional graphical effects
- More enemies and bosses
